# 터보 다운힐 레이서

HTML, CSS, JavaScript만으로 만든 종스크롤 레이싱 게임입니다. 브라우저에서 `index.html`을 열거나 로컬 정적 서버로 실행할 수 있습니다.

## 실행 방법

```bash
python3 -m http.server 4173
```

그 다음 `http://127.0.0.1:4173/`에 접속하세요.

## 조작법

- `←` / `→` 또는 `A` / `D`: 차선 이동
- `Space`: 일시정지 또는 재개
- 모바일/터치 화면: 트랙의 왼쪽 또는 오른쪽을 터치해 이동

## 게임 규칙

- 장애물 차량과 충돌하면 게임이 종료됩니다.
- 별 코인을 모으면 추가 점수를 얻습니다.
- 시간이 지날수록 속도가 빨라져 난이도가 올라갑니다.
